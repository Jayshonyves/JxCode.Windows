# JxCode.Windows
基于Win32API编写的C#windows操作类库。可以使用在控制台、链接库、以及Unity等windows平台的框架下。

## 核心模块
INI配置文件读写  
鼠标模拟控制（位置，按键模拟）  
键盘模拟控制（按键模拟）  
窗体控制（句柄，位置，大小，标题，类名，图标，层级，焦点，最大化，最小化，还原，窗体消息）  
注册表数据储存  
信息对话框  
文件对话框（打开，储存）  
托盘程序NotifyIcon  
MCI多媒体接口（播放音频，视频）  
进程句柄模块（输入，输出，退出事件）
